## 명명법

* Lower Camel Case
  * 함수, 메소드, 변수, 상수
  * ex) someVariableName
* Upper Camel Case
  * type 이름 (클래스, 구조체, enum, extension … )



## 콘솔로그 남기기

* print 함수 : 단순 문자열 출력
* dump 함수 : 인스턴스의 자세한 설명(description 프로퍼티)까지 출력



## 문자열 보간법

* String Interpolation
* 프로그램 실행 중 문자열 내에 변수 또는 상수의 실질적인 값을 표현하기 위해 사용
* \\()
  * ex) print("안녕하세요! 저는 \\(age)살입니다")
